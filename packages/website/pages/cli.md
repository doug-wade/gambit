# @tybalt/cli

## Getting Started

Use `@tybalt/cli` with `npx`

```shell
$ npx @tybalt/cli build
```

## Commands

### build

Builds an application

```shell
$ npx @tybalt/cli build
```

### scaffold

Scaffolds a new application

```shell
$ npx @tybalt/cli scaffold
```

### serve

Serves an application

```shell
$ npx @tybalt/cli serve
```
