tybalt.org

The docs site for Tybalt. We build the website and deploy the documentation website with every commit in the Tybalt monorepo as part of our continuous integration pipeline. Be careful when looking through the website's source code, as it may exercise features that aren't available in public releases yet.

I Want To Be A Fan

A simple USWNT fan site that displays statistics from American Soccer Analysis using itscalledsoccer. A great example for how to set up building a static site using the eleventy plugin.
